    mojo Module

    Defined in: packages/mojo.js/lib/registeredClasses.js:3

    Main entry point to your application. This is where everything is initialized.

    This module provides the following classes:

    • Animator
    • Application
    • BaseView
    • BindableCollection
    • BindableObject
    • Binding
    • BindingTransformer
    • EventEmitter
    • ListView
    • Mojo
    • RegisteredClasses
    • StateView

    This module is a rollup of the following modules:

    • mojo-core
      created when the second parameter on bind(property, listener) is an object.
    • mojo-views
      ## Usage
      `javascript var SubView = mojo.View.extend({ name: "craig" }); var view = new SubView(); console.log(view.get("name")); //craig ` ## Sections Property The sections property allows you to define sub-views. `javascript var PagesView = mojo.View.extend({ sections: { header: require("./headerView"), content: require("./contentView") } }) ` ## Events Property
